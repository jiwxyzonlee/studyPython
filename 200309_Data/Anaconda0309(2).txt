: 전체 기여도 파악할 때 주로 이용
- 하나의 열을 가지고 작성
- 메소드 이름은 pie
- 옵션으로는 explode 에 실수 리스트를 대입할 시 중앙에서 대입한 숫자만큼 떨어져서(조각) 그려짐
->> 0.0 ~ 1.0 사이
- autopct 에 숫자 서식을 설정하여 백분율 표시 가능
->> '%1.1f%%' (소수 첫째자리까지의 백분율)


8. boxplot 그래프
: 데이터의 분포를 확인할 때 사용하는 그래프
- 상자의 중앙선이 중간값, 상자의 좌우가 25%, 75% 에 해당하는 값
->> 양끝선이 아래쪽 0.75% 정도 위끝선은 99.75% 정도에 해당하는 값
->> 작은 원으로 표현되는 극단치(outlier)를 판단
- 중앙선과 양끝단의 선, 상자의 선을 보고 데이터의 분포를 예측함
->> 한쪽으로 치우치면 정규분포가 아닌 형태로 판정


9. 면적 그래프
: 선 그래프의 하단에 색상을 칠한 그래프
- 목적은 막대그래프와 유사
- fill_between 이라는 메소드를 이용



** seaborn 패키지
: matplotlib을 기반으로 하여 여러 테마와 통계용 차트가 추가된 패키지 
- numpy나 pandas의 자료구조는 그대로 사용

seaborn: https://seaborn.github.io/


1. regplot 이나 lmplot이라는 함수를 이용하여 데이터의 분포와 1차 다항식 그래프 동시 출력 가능
->> 1차 다항식은 2개의 컬럼의 회귀식


2. 다변량 데이터의 분포를 출력
- 히스토그램을 이용하는 경우가 많음
->> 쉽다는 장접이 있지만 불연속하게 그려짐
- 히스토그램 대신에 커널밀도함수(kde)를 적용해서 연속적으로 그려 파악하는 경우가 있음
- seaborn의 distplot이라는 함수를 이용하면 커널 밀도 함수를 적용한 그래프와 히스토그램을 같이 출력할 수 있음


3. heatmap
: 크로스 테이블을 시각화할 때 주로 이용
- 항목 2개를 적용해서 분류한 후 데이터를 출력하여 시각화
- Dataframe에서 pivot이라는 함수를 이용하여 2개로 분류하고 계산할 항목과 함수를 지정
->> 함수 생략 시 합계로 적용됨
- heatmap 함수에 데이터를 대입하면 시각화됨


4. violinplot
- boxplot은 범주형 데이터 값들의 상한 혹은 하한, 4분위 수 등을 파악
- violinplot은 사각형 대신에 커널밀도함수를 적용해서 데이터 개수도 어느 정도 파악이 가능함


5. 데이터 개수 출력
- stripplot, swarmplot() 등이 제공


6. 막대 그래프 - barplot


7. 히스토그램 - countplot


8. 데이터프레임의 모든 컬럼의 상관관계를 출력 - pairplot


9. 상관관계를 출력하고 히스토그램까지 출력 - jointplot

**folium
: 지도나 단계구분도를 위한 시각화 패키지
- 자바스크립트 기반으로 interactive(동적 변화) 하게 시각화
- 기본 패키지가 아니므로 설치 필요
- jupyter notebook가 크롬에서 실행 중이라면 바로 출력 가능
- 다른 IDE 사용 시 save('파일 경로.html')로 저장한 후 브라우저에서 확인해야 함
->> IE에서는 자바스크립트 문제로 출력 불가

- 단계구분도를 만들 때는 각 지역의 경계에 대한 json 데이터가 존재해야 함
->> 한국 데이터는 southkorea-maps에서 확인 가능


1. 지도 생성
folium.Map(locations = [위도, 경도], zoom_start = 확대축소 배율)


2. 마커 생성
folium.Marker(locations = [위도, 경도], popup = 출력될 문자열, icon = 이미지.add_to(지도 객체)


3. 출력
- 주피터 노트북: 지도 객체
- 다른 IDE: 지도객체.save('파일 경로')
