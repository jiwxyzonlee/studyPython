** 시각화
- 데이터의 형태를 파악하거나 보고서를 만들 목적으로 그래프나 지도 등을 출력하는 작업
- matplot(기본 시각화), pandas, seaborn(시각적인 효과가 조금 더 뛰어남), folium(지도, 단계 구분도) 등을 주로 이용


1. 꺾은 선 그래프 - plot
: 날짜나 시간에 따른 변화량 또는 2개의 컬럼을 가지고 데이터의 분포를 알아볼 때 많이 이용


2. 하나에 2개의 그래프 그리기

import matplotlib.pyplot as plt

객체 = plt.figure(figsize = (가로크기, 세로크기))
변수1 = 객체.add_subplot(행의 개수, 열의 개수, 자신의 번호)
변수2 = 객체.add_subplot(행의 개수, 열의 개수, 자신의 번호)

변수1.plot(옵션 ...)
변수2.plot(옵션 ...)


3. 하나의 영역에 여러 개의 그래프 그리기
: 동일한 영역에서 그리는 함수를 여러 번 호출


4. 막대 그래프
: 빈도수를 비교할 때 주로 이용
- bar(수직), barh(수평) 메소드로 그림
- 하나의 영역에 2개의 막대그래프를 그릴 때는 첫번째 막대 그래프의 width를 조정
->> width가 1이면 다음 데이터와 붙어서 출력됨
->> 2개일 시 0.5 이하로 설정해서 그리고 3개일 시 0.33이하로 설정


5 Histogram
: 각 구간의 데이터 개수를 파악하고자 할 때 사용하는 그래프


6. 산포도
: 컬럼 2개의 관계를 파악하기 위해서 생성
- 상관 여부나 그룹핑할 개수 등을 파악하고자 할 때 먼저 그려보면 효과적
->> 군집 여부와 성향에 따라 개수 나누어 그룹화하는 게 좋음
- colorbar를 이용해서 산포도에 그려지는 데이터의 색상에 크기 적용 가능
